CraftyJsHelloWorld

Hello world crafty.js game

To run the integrated web server is necessary to have installed https://nodejs.org/.

Install bower if not installed:

npm install bower -g

Install dependencies:

cd front
bower install
cd ..\server
npm install

Run server:

node server/server.js
